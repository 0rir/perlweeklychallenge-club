# Swap Nibbles
## Java Solution

To compile the code run
```java
javac SwapNibbles.java
```
To run
```java
java SwapNibbles
```
### Example
```bash
/perlweeklychallenge-club/challenge-119/ziameraj16/java$ javac SwapNibbles.java
/perlweeklychallenge-club/challenge-119/ziameraj16/java$ java SwapNibbles
Enter the number
101
86
/perlweeklychallenge-club/challenge-119/ziameraj16/java$ java SwapNibbles
Enter the number
18
33
```
