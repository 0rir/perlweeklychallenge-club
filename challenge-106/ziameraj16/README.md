# Maximum Gap
## Java Solution

To compile the code run
```java
javac MaximumGap.java
```
To run
```java
java MaximumGap
```

```bash
/perlweeklychallenge-club/challenge-106/ziameraj16/java$ javac MaximumGap.java
/perlweeklychallenge-club/challenge-106/ziameraj16/java$ java MaximumGap

```
