# Odd Number
## Java Solution

To compile the code run
```java
javac OddNumber.java
```
To run
```java
java OddNumber
```
### Example
```bash
/perlweeklychallenge-club/challenge-130/ziameraj16$ java OddNumber
Input comma separated values
2,5,4,4,5,5,2
5
/perlweeklychallenge-club/challenge-130/ziameraj16$ java OddNumber
Input comma separated values
1,2,3,4,3,2,1,4,4
4
```
