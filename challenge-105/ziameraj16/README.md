# Nth Root
## Java Solution

To compile the code run
```java
javac NthRoot.java
```
To run
```java
java NthRoot
```
### Example
```bash
/perlweeklychallenge-club/challenge-105/ziameraj16/java$ javac NthRoot.java
/perlweeklychallenge-club/challenge-105/ziameraj16/java$ java NthRoot
Enter value of N
5
Enter value of k
248832
12.00
```
