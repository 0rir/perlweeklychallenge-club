# Canonical Path
## Java Solution

To compile the code run
```java
javac CanonicalPath.java
```
To run
```java
java CanonicalPath
```
### Example
```bash
/perlweeklychallenge-club/challenge-112/ziameraj16/java$ java CanonicalPath
Enter a string
/a/
/a
/perlweeklychallenge-club/challenge-112/ziameraj16/java$ java CanonicalPath
Enter a string
/a/b//c/
/a/b/c
/perlweeklychallenge-club/challenge-112/ziameraj16/java$ java CanonicalPath
Enter a string
/a/b/c/../..
/a
```
