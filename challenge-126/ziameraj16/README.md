# Count Numbers
## Java Solution

To compile the code run
```java
javac CountNumbers.java
```
To run
```java
java CountNumbers
```

```bash
/perlweeklychallenge-club/challenge-126/ziameraj16/java$ javac CountNumbers.java
/perlweeklychallenge-club/challenge-126/ziameraj16/java$ java CountNumbers
```
